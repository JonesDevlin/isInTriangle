# isInTriangle

Problem:
Given an array of points that represent the 3 vertices of a triangle, and a point K, return true if K is inside the triangle.

Example:

    let triangle = [ [0,0], [0,3], [4,0] ]
  
    isInTriangle(triangle, [2,1])
    > true

    isInTriangle(triangle, [3,2])
    > false
